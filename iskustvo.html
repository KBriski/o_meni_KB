  • Kao učiteljica razredne nastave radim u osnovnoj školi već 9 godina (u razredu, u produženom boravku, kao učiteljica informatike)

  • Upisujem studij Informatike kako bih mogla ostati u školi i predavati učenicima informatiku ili krenuti u novome smjeru.

Iskustvo u informatici

  • Kao studentica, dobila sam široka teorijska znanja i sada mi ostane odabrati ono što me zanima i krenuti s radom.
  • Sviđa mi se razvoj web stranica i aplikacija kao i mobilnih aplikacija pa bih u tome smjeru i htjela napredovati
  • Tijekom studija koristila sam VS Code za razvoj koda koristeći PHP, HTML, CSS, JavaScript, Dart, Python za izradu projekata: mobilna app za praćenje osobnih troškova, web stranica putničke agencije s mogućnošću rezervacije putovanja
  • Mobilna aplikacija Troškovnik je Flutter aplikacija za Android okruženje koja omogućuje praćenje troškova korisnika putem jednostavnog sučelja i pohrane podataka u lokalnu SQLite bazu podataka. Korišten je Dart programski jezik.
  • Web stranica Putnička agencija je dinamička web stranica u kojoj je glavni jezik skriptni PHP uz korištenje HTML, CSS i JavaScript. Aplikacija koristi MySQL relacijsku bazu podataka koja je uređivana putem web alata phpMyAdmin.
  • Web aplikacija za produženi boravak izradila sam za završni rad. Glavni jezik za izradu aplikacije je skriptni PHP uz korištenje HTML, CSS i JavaScript. Aplikacija koristi MySQL relacijsku bazu podataka koja je uređivana putem web alata phpMyAdmin.
  • Koristila sam XAMPP lokalno okruženje web poslužitelja za brzo i jednostavno testiranje aplikacije te uočavanje nedostataka prilikom stvaranja koda na vlastitom računalu
  • Najradije koristim VS Code. Za statičke web stranice koristim HTML, CSS i JS, a za dinamičke web stranice/app koristim PHP, MySQL relacijsku bazu koju uređujem putem web alata phpMyAdmin, a sve lokalno testiram koristeći XAMPP.
