Iskustvo u školi

  • Kao učiteljica razredne nastave radim u osnovnoj školi već 9 godina (u razredu, u produženom boravku, kao učiteljica informatike)
  • Zadnje 3 godine radim u istoj školi u produženom boravku s prekidima preko ljeta.
  • Upisujem studij Informatike kako bih mogla ostati u školi i predavati učenicima informatiku ili krenuti u novome smjeru.

Iskustvo u informatici

  • Kao studentica stekla sam široka teorijska znanja iz područja informatike, a kroz praktične projekte otkrila sam poseban interes za razvoj web i mobilnih aplikacija. U budućnosti želim dalje razvijati vještine upravo u tom smjeru.
  • Tijekom studija koristila sam razvojno okruženje Visual Studio Code te radila u sljedećim programskim jezicima: HTML, CSS, JavaScript, PHP, Dart i Python. Svoje znanje primijenila sam u izradi konkretnih projekata:
    • 1) Mobilna aplikacija Troškovnik: Izrađena pomoću Fluttera i Dart jezika za Android uređaje. Aplikacija omogućuje jednostavno praćenje osobnih troškova uz pohranu podataka u SQLite lokalnu bazu podataka.
    2) Web stranica Putnička agencija: Dinamička web aplikacija izrađena u PHP-u, uz korištenje HTML-a, CSS-a i JavaScripta. Koristi MySQL relacijsku bazu podataka, uređivanu putem alata phpMyAdmin. Omogućuje pregled i rezervaciju putovanja.
    3) Web aplikacija za produženi boravak: Izrađena kao završni rad. Također je riječ o dinamičkoj web aplikaciji u kojoj je korišten PHP, uz HTML, CSS i JS. Za pohranu podataka koristi se MySQL, a baza je upravljana putem phpMyAdmina.
  • Tijekom stručne prakse izradila sam osobne projekte koji imaju primjenu u svakodnevnom životu: digitalni životopis i web stranicu za produženi boravak. Također sam doradila prijašnji projekt o putničkoj agenciji, poradila sam na responzivnosti.
    • 1) Digitalni životopis je statična web stranici koja predstavlja moj osobni portfolio. Uključuje pregled vještina, obrazovanja i dosadašnjeg iskustva. Korištene tehnologije: HTML5, CSS3 (uključujući Bootstrap za responzivni dizajn) i JavaScript (za dark/light način rada te dinamičko učitavanje headera i footera).
    2) Web stranica Produženi boravak je informativna web stranica namijenjenu roditeljima i učenicima koji pohađaju produženi boravak. Sadrži osnovne informacije, dnevni raspored, pravila ponašanja te galeriju (povezanu s Wakeletom) i obavijesti koje se ručno ažuriraju u HTML-u. Projekt je osmišljen kao praktičan alat koji se može koristiti u školskom okruženju.
  • Za razvoj i lokalno testiranje svih aplikacija koristila sam XAMPP okruženje, što mi je omogućilo brzo uočavanje grešaka i testiranje funkcionalnosti.
